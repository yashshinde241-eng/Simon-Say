# Simon Says Game

A classic memory game built with vanilla JavaScript, HTML, and CSS. Test your memory by repeating increasingly complex sequences of colored button presses!

## 🎮 How to Play

1. Press the **Enter** key to start the game
2. Watch the sequence of button flashes
3. Click the buttons in the same order as the sequence
4. Each level adds one more button to the sequence
5. If you make a mistake, the game ends and shows your score
6. Try to beat your high score!

## ✨ Features

- **Progressive Difficulty**: Each level increases the sequence length
- **High Score Tracking**: Your best score is saved during the session
- **Visual Feedback**: Buttons flash when pressed and during sequences
- **Error Animation**: Shake effect when you make a mistake
- **Responsive Design**: Centered layout that works on different screen sizes
- **Keyboard Controls**: Start the game with Enter key

## 🛠️ Technologies Used

- **HTML5**: Structure and layout
- **CSS3**: Styling and animations
- **Vanilla JavaScript**: Game logic and interactivity

## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)

### Running the Game

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Press Enter to start playing!

### Files Structure

```
├── index.html    # Main HTML file
├── style.css     # Game styling and animations
└── app.js        # Game logic and event handlers
```

## 🎨 Game Mechanics

- The game generates a random sequence of 4 colored buttons
- Players must memorize and repeat the sequence exactly
- Correct sequences advance to the next level
- Incorrect presses end the game
- High score persists until the browser is refreshed
